# Galerie Miniaturbilder Gridvorschau  
## WORK IN PROGESS

## Bildergalerie mit Thumbnail und Bildercounter  

Ein bis fünf Vorschaubilder werden in einem Grid dargestellt. Bei höherer Bildanzahl als das Vorschaugrid erlaubt, wird die Anzahl der zusätzlich vorhandenen Bilder als Zahl dargestellt.  
Bei Klick wird die ganze Galerie in der Lightbox [Glightbox wird empfohlen](https://github.com/inspiredminds/contao-glightbox) angezeigt.
